# Introducción al Análisis Genómico con SNPs

Curso introductorio para análisis de diversidad genética y estructura poblacional

Este repositorio contiene materiales, scripts y recursos para un curso práctico de análisis genómico utilizando datos SNP en el contexto de antropología biológica, bioinformática y genética humana. El enfoque principal es enseñar análisis estándar de genética poblacional utilizando herramientas ampliamente utilizadas en investigación contemporánea.

⸻

## Objetivos del curso
	•	Introducir conceptos básicos de genética de poblaciones aplicada a datos SNP.
	•	Guiar el procesamiento y análisis de datos genotípicos.
	•	Enseñar el uso de herramientas clave bajo entorno Linux (Ubuntu).
	•	Interpretar resultados en contexto antropológico y genómico.

⸻

## Estructura del repositorio

- data/        # Datasets de ejemplo
- scripts/     # Scripts de analisis
- results/     # Resultados generados
- docs/        # Material de apoyo y slides
- README.md

## Notas éticas

Este material está diseñado para fines educativos. Si trabajas con datos genómicos de poblaciones humanas, considera:

	•	Consentimiento informado y contexto histórico.
  
	•	Sensibilidad cultural y comunitaria.
  
	•	Interpretación no determinista ni biologizante.
